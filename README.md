# 汉语道（ハンユゥダオ）― 日本語話者のための中国語総合学習アプリ

日本語話者に最適化された、試験対策（HSK・中国語検定 / CEFR A1〜B2）と実践会話を両立する AI 搭載中国語学習アプリです。

## 起動方法

ビルド不要。`index.html` をブラウザで開くだけで動きます。

```
open index.html
```

学習データはブラウザの localStorage に自動保存されます（設定画面からエクスポート/リセット可能）。

## 機能一覧

| 機能 | 内容 |
|---|---|
| 🗺️ ロードマップ | CEFR A1→B2 / HSK1〜5 / 中検準4級〜準1級 対応の4ステージ。達成度を自動計測 |
| 📚 単語 | **2003語**（HSK1〜5準拠・全語ピンイン&日本語訳つき、コア120語は例文つき）。SRS（SM-2忘却曲線）フラッシュカード、中→日 / 日→中 / リスニングクイズ、一覧検索 |
| ✏️ 文法 | **31項目**。アスペクト（了・过・着・在）、把/被構文、各種補語、命令・仮定・使役・離合詞・強調構文など。**日本語話者のよくあるミス**と**瞬間変換ドリル**付き |
| 📖 読解 | 短文→中文→長文の**15篇**（日常・旅行・文化・ニュース・試験形式）。新出単語・文法メモ・音声・内容理解問題・要約タスク・AI質問 |
| 🎧 リスニング | ディクテーション（自動判定・diff表示）とシャドーイング。再生速度 0.5〜1.5x |
| 💬 会話 | **18シーン**（挨拶〜タクシー・駅・美容院・薬局・電話予約・就職面接まで）。通し再生、ロール練習（B役ブラインド）、**AIロールプレイ** |
| 🗣️ 発音 | 四声・軽声、**音節表（396音節×四声を代表漢字つきで再生）**、日本人が苦手なピンイン解説、変調ルール、声調聞き取りクイズ、**🎤発音チェック（音声認識で「音」と「声調」を分けて採点）** |

| 🤖 AI | AI教師（添削・解説・宿題）、AIコーチ（学習データ分析・計画提案）、AI自由会話（レベル自動調整・誤り訂正） |

## AI機能のセットアップ

1. [console.anthropic.com](https://console.anthropic.com/) で API キーを取得
2. アプリの「⚙️ 設定」→ APIキーを入力して保存（この端末の localStorage にのみ保存されます）
3. モデルは `claude-sonnet-5`（推奨）/ `claude-opus-4-8` / `claude-haiku-4-5` から選択

AIキーがなくても、AI以外の全機能（SRS・クイズ・読解・リスニング・発音・ルールベースの学習コーチ提案）は動作します。

## 音声について

Web Speech API（OS内蔵TTS）を使用します。macOS で中国語音声が出ない場合は
「システム設定 → アクセシビリティ → 読み上げコンテンツ → システム音声 → 管理」から
中国語ボイス（Tingting 等）を追加してください。

## 設計メモ（拡張ポイント）

- `data.js` … コア学習コンテンツ（例文つき単語・文法・読解・会話・発音・ロードマップ）
- `vocab-extra.js` … HSK準拠の追加語彙1883語（自動生成＋日本語訳・ピンイン監修）
- `grammar2.js` … 追加文法15項目
- `content-extra.js` … 追加読解9篇・追加会話6シーン
- `pinyin-data.js` … 音節×声調→代表漢字（1096組）・漢字→ピンイン対応（3034字、発音採点用）
- **各ファイルに追記するだけでコンテンツ拡張可能**
- `app.js` … SRSエンジン（SM-2）、クイズ、統計、AI連携、ルーター
- 将来拡張の想定：発音採点（Web Speech Recognition / 音声解析API）、繁体字対応、書き取り（手書き認識）、命令・仮定表現などの文法追加、サーバー同期
